# Lawson's Skills

Install by running:

```bash
bunx skills add lawsonoates/skills
```

## Skills

| Skill | Description |
|-------|-------------|
| `setup-monorepo` | Scaffold a new Bun monorepo with Turbo, tsgo, and Ultracite. Use when setting up or initializing a Bun workspace. |
| `write-script` | Write high-quality Bun scripts — workflow pipelines (run once, exit) and service scripts (long-running processes). Use for root automation, dev tooling, deploy pipelines, process supervisors, and local servers. |
